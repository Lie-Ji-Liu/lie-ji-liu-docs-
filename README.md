# 个人长剑笔记与框架（GitHub）

**——记录我的探索之旅，诚求 HEMA 大佬批评指正**

---

这段时间我基本一有空就投入到这件事上，最近才完成了 0.1.0 版本。  
总的来说，这是一份融合**稳态**与**爆发态**、强调灵活适应、安全导向、控制优先的长剑探索笔记，旨在实现低风险、高效对抗。

这份文档分为六个部分：**架势、步伐、动作链、心法、学员阶段对照、应对不同对手类型**。  
没有训练法，是因为我并不想标新立异创立流派，只是单纯记录自己的探索过程并分享出来。  
看到这些内容的朋友，可以把它当作茶余饭后的谈资或兴趣读物，吸收一些思路精华当然欢迎，但也请务必带着辩证性的思维。

### 架势篇
主要围绕**稳态耳卫犁基**展开——这是一个融合犁势（Pflug）和吊势（Hängen）特点的中继位，负责控中线、控距离、读反应、诱导，在保留进攻可能性的同时随时准备防御反击，并支持灵活切换。  
作为中继位，可根据情况切入犁势、牛势、长点势、吊势、屋顶势。  
另有一个非常规的**爆发态怒猎势**，抓住机会后先控剑、再精准猎杀人。稳态与爆发态互补。

### 步伐篇
罗列了几乎所有常见战术意图对应的步伐实现方式，以及耳卫犁基和怒猎势的专属步伐。

### 动作链篇
讲的是怎样**灵活且不亏**地打，而不是强行得分。面对对手制造的问题，用低风险方式解决；同时包括怒猎势的**压迫 → 控制 → 终结 → 撤离**完整流程。

### 心法篇
- **四个时机**：观察对手、锁死对手、一击直取、安全撤离  
- **三条戒律**：不换命、不露中线、不迟疑  
总的来说就是**打聪明的架，不做亏本买卖**——不上头硬拼，也不犹豫败北。  
怒猎势的主动压迫也必须严格遵循这四个时机和三条戒律。

### 学员阶段划分篇
- **新手期**：动作果断、步伐灵活、基本距离感与线感、基础格挡与反应；禁止心理博弈和复杂防反（基础能力不足）。  
- **进阶期**：控制中线与距离、读反应，此时防反才可靠（读到的反应是真实的）。  
- **高手期**：节奏感极强，能压缩局面限制对手选择，在脑中预设多条线路，架势流动自如。  
- **顶峰阶段**：高度兼容性，吸收各体系精华于一身，集多家高手所长。这是一条永无止境的路，如天上启明星指引方向。

### 应对不同对手篇
通用流程：观察 → 试探判断类型 → 确保自身安全 → 积累优势 → 发现胜机（往往仅0.5–1拍） → 执行取胜。  
对手类型：诱导陷阱型、快速进攻型、防守反击型、缠斗型、游走型。

> **注意**：这些内容**并不保证在所有规则下都有效**。  
> 我个人对“稳态中继位”在高强度对抗中的表现，以及“双态框架”是否能比肩传统体系，也抱有很大不确定性。非常欢迎基于实战或历史体系的反例与批评。
